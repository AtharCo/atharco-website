# AtharCo Website

**AI-Powered Infrastructure for Local Economies**

The official website for AtharCo Holdings — a data-driven technology holding company building AI-native platforms for MENA.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Font:** IBM Plex Sans Arabic
- **Animation:** Framer Motion
- **Hosting:** Vercel
- **Design:** "Autumn Technical" brand identity

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Deep Forest | `#283618` | Primary text, headers |
| Olive | `#606c38` | Secondary text, borders |
| Warm Gold | `#bc6c25` | CTAs, accents |
| Cream | `#fefae0` | Background |
| Light Olive | `#e9edc9` | Surface, tags |

## Getting Started

```bash
npm install
npm run dev
```
